Perguntas Frequentes

Entenda os detalhes da nossa implementação tecnológica.

A estrutura foi montada para operar primariamente sem conexão (Offline) para visitas em campo. As informações coletadas presencialmente são armazenadas no próprio aparelho móvel. Quando a equipe retorna para um local com rede ativa, tudo é enviado automaticamente para a base de dados principal.

Sim, a arquitetura é totalmente modulável. É possível criar novas áreas para abranger diferentes programas sociais, benefícios eventuais e outras necessidades específicas da secretaria, mantendo tudo integrado em um só lugar e altamente personalizável para cada necessidade.

O foco foi utilizar soluções nativas e plataformas corporativas já acessíveis. Ao aproveitar ferramentas robustas disponíveis no mercado, conseguimos entregar altíssima qualidade técnica e inovação sem gerar dependência de licitações custosas.
